# Phase 3 구현 보고서

## 범위

Phase 3에서는 세계관 생성, 조회, 수정, 삭제, 공개 탐색, 태그, 공유 관리 흐름을 실제 Prisma 데이터와 연결했다. 화면은 `docs/ui/06`, `08`, `09`, `10`, `11`, `27` 목업을 기준으로 구성했다.

## 구현 내용

### 세계관 CRUD

- 세계관 생성 및 수정 폼 구현
  - 제목
  - URL 슬러그
  - 커버 이미지 URL
  - 공개 범위
  - 장르
  - 태그
  - Markdown 소개
- Server Action마다 세션과 세계관 권한을 다시 검증
- 생성 시 OWNER 멤버십을 함께 생성
- 수정 후 상세 화면으로 이동 및 관련 경로 갱신
- 내 세계관 목록에서 선택 삭제 모드와 확인 모달 구현
- 삭제는 소유자만 가능한 논리 삭제로 처리

### 태그

- 입력값의 `#` 제거
- 공백 기준 분리
- 소문자 정규화 및 중복 제거
- 세계관별 태그 upsert
- 세계관 수정 시 기존 `world_tags` 연결 갱신
- 공개 탐색에서 제목, 소개, 장르, 태그 통합 검색

### 공개 탐색

- PUBLIC 세계관만 조회
- 인기순 `viewCount desc`, 최신순 `updatedAt desc`
- 페이지당 15개 페이지네이션
- 검색 초기화, 빈 결과 화면, 정렬 상태 표시

### 세계관 상세

- 커버 이미지 또는 기본 그라디언트 커버
- 모바일 반응형 커버 비율과 웹페이지 URL의 Open Graph 이미지 해석
- Markdown 소개
- 캐릭터, 소속, 창작물 개수
- 캐릭터, 소속, 창작물 가로 카드 목록
- 편집 권한 사용자에게 캐릭터, 소속, 창작물 추가 CTA 제공
- 관계 그래프 미리보기 영역
- 데이터가 없는 섹션의 빈 상태
- OWNER, ADMIN, EDITOR 역할에 따른 수정 및 공유 버튼 노출

### 공유 관리

- OWNER, ADMIN, EDITOR, VIEWER 표시
- 이메일 초대 모달
- 가입 사용자는 멤버십 upsert
- 미가입 이메일은 대기 초대 생성
- 권한 변경
- 멤버 공유 해제
- 대기 초대 취소
- OWNER 역할은 소유자 행에서 고정해 우발적인 소유권 변경을 방지

### 공통 개선

- 권한 유틸에서 `World.ownerId`를 직접 소유자로 인정하도록 보완
- 로그인 상태에 따라 헤더의 로그인/로그아웃 CTA 전환
- Markdown Viewer의 제목, 목록 렌더링 보완
- 목록 카드 소개에서 Markdown 기호 제거

## UI 검증

검증은 요청에 따라 `npm run build`, `npm run lint`를 실행하지 않고 `localhost:3000`을 Chrome으로 직접 열어 진행했다.

- `/worlds/explore`
  - HTTP 200
  - 오류 오버레이 없음
  - 검색, 정렬, 빈 결과 패널 확인
- 회원가입 및 로그인
  - 브라우저에서 신규 계정 생성 후 로그인 성공
- `/worlds/new`
  - PUBLIC 세계관 생성
  - Markdown 소개 및 태그 저장
  - 생성 후 상세 화면 이동
- `/worlds/[worldId]`
  - 커버, 소개, 요약 지표, 빈 리소스 섹션 렌더링
- `/worlds/[worldId]/edit`
  - 제목 수정 및 상세 화면 반영
- `/worlds/[worldId]/share`
  - VIEWER 이메일 초대 생성 및 초대 대기 표시
- `/worlds/explore?q=%23판타지`
  - 태그 검색으로 생성한 세계관 노출
- `/worlds`
  - 390px 모바일 레이아웃 가로 오버플로 없음
  - 선택 삭제 모드, 확인 모달, 논리 삭제 성공

## 데이터베이스

브라우저 검증 중 로컬 PostgreSQL에 Phase 1 스키마가 적용되지 않아 `public.worlds` 테이블 오류가 확인되었다. 기존 `prisma/schema.prisma`를 `prisma db push`로 로컬 데이터베이스에 동기화한 뒤 모든 Phase 3 브라우저 흐름을 검증했다.

- Next.js 오류 오버레이: 없음
- 데스크톱 및 모바일 주요 화면 렌더링: 정상
- 세계관 생성, 조회, 수정, 삭제: 정상
- 태그 저장 및 검색: 정상
- 이메일 초대 및 공유 관리: 정상
