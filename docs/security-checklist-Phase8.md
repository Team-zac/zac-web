# Phase 8 권한 및 보안 체크리스트

## Server Action 권한 점검

- 세계관 생성: 로그인 사용자만 가능
- 세계관 수정: 세계관 편집 권한 필요
- 세계관 삭제: 소유자만 가능, 논리 삭제
- 세계관 공유 관리: 소유자 또는 관리자 권한 필요
- 캐릭터 생성/수정/삭제: 세계관 편집 권한 필요, 논리 삭제
- 소속 생성/수정/삭제: 세계관 편집 권한 필요, 삭제 시 캐릭터-소속 연결 제거 및 하위 소속 parent 해제
- 관계 생성/수정/삭제: 세계관 편집 권한 필요, 논리 삭제
- 창작물 생성/수정/삭제: 세계관 편집 권한 필요, 창작물 삭제는 논리 삭제
- 챕터 추가/수정/삭제/게시: 창작물이 속한 세계관 편집 권한 필요, 챕터 삭제는 논리 삭제

## 공개/비공개 조회 점검

- 공개 세계관 탐색은 `PUBLIC` 세계관만 조회
- 공개 캐릭터 탐색은 `PUBLIC` 캐릭터와 `PUBLIC` 세계관만 조회
- 공개 창작물 탐색은 `PUBLIC` 및 `PUBLISHED` 창작물과 `PUBLIC` 세계관만 조회
- 내 목록은 소유하거나 공유받은 세계관 경계 안의 리소스만 조회
- 상세/편집 조회에서 인증 또는 권한 실패 시 존재 여부가 드러나지 않도록 `null` 반환 후 `notFound` 흐름으로 처리

## Markdown 렌더링

- 공통 `MarkdownViewer`에 sanitize 적용
- 창작물 리더 Markdown 파서에 sanitize 적용
- HTML 태그, 제어 문자, `javascript:`, `data:`, `vbscript:` 프로토콜 제거
- React text rendering을 유지해 HTML을 직접 삽입하지 않음

## 오류 메시지 표준화

- 인증 필요: `로그인이 필요합니다.`
- 권한 없음: `접근 권한이 없습니다.`
- 존재하지 않음: `대상을 찾을 수 없습니다.`
- Prisma 검증/DB 오류: 일반화된 처리 메시지
- 알 수 없는 오류: `요청을 처리하는 중 오류가 발생했습니다.`
